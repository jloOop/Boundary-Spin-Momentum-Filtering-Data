# Time-evolution wave-function GIFs

This folder is for GIFs and PNG frame collections showing the time evolution of the probability density.

The GIFs are generated from saved density snapshots

```text
rho_prob_t*.npy
```

using

```text
notebooks/Spin_Gif.ipynb
```

Recommended direct files:

```text
omega_001_midplanes.gif
omega_100_midplanes.gif
omega_200_midplanes.gif
omega_300_midplanes.gif
omega_001_contour_xy.gif
omega_300_isosurf.gif
```

For large files, upload ZIPs to GitHub Releases and link them in [`Links-to-Gifs.md`](Links-to-Gifs.md).

## Suggested caption

The GIFs show detector-present spinor-ABC evolution for a Gaussian longitudinal wave packet in a harmonic waveguide. Increasing `omega` narrows the transverse ground state and enhances the delayed sector of the roof-flux signal.
