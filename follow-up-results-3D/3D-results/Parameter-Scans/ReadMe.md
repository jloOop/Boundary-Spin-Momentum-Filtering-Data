# Parameter scans

This folder is for scans showing how source and detector parameters change the finite-window coefficients and late tails.

Recommended subfolders:

```text
kappa-scan/
sigma-z-scan/
kz-scan/
large-box-check/
```

The main message is that the coefficients depend on the source, detector parameter, geometry, and observation window, while the local origin of the confinement dependence remains the boundary scale `|xi| ~ sqrt(omega)`.

Suggested summary tables:

```text
kappa_scan_summary.csv
sigma_z_fit_coefficients.csv
kz_scan_summary.csv
large_box_check_summary.csv
```
