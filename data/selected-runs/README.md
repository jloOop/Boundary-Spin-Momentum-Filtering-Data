# Selected runs

Each selected run should contain enough to reproduce one figure or GIF.

Recommended minimal files:

```text
constants.npz
prob_times.npy
total_probs.npy
det_rate_toprow.npy
summary.json
```

For density GIFs:

```text
rho_prob_t*.npy
```

For trajectory plots:

```text
bohmian_traj_selected.npy
bohm_times.npy
traj_indices.npy
bohm_arrived_mask_selected.npy
bohm_t_hit_selected.npy
```
